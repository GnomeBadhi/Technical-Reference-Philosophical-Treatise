# Sovereign Kernel Demo

This is a fully client-side demonstration of the Sovereign Kernel.

## Run It

1. Clone the repo  
2. Enable GitHub Pages (Settings → Pages → Source: `/docs`)  
3. Visit:https://github.io/GnomeBadhi/Technical-Reference-Philosophical-Treatise/new/main/docs/DEMO

4. The demo runs entirely in the browser using:
- HTML5
- Canvas 2D
- WebGL (raw)
- Zero dependencies

## What It Shows

- Intent parsing  
- Operator selection  
- State transitions  
- Delta computation  
- 2D visualization  
- 3D invariant space  

This is a public, inspectable proof-of-work for the kernel.
